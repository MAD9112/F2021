# 2. Boot Camp

From Apple website

Boot Camp is a utility that comes with your Mac and lets you switch between macOS and Windows. Download your copy of Windows 10, then let Boot Camp Assistant walk you through the installation steps for Intel-based Macs.

We will use Boot Camp and the ISO file that you downloaded.

Do I need a USB flash drive to install Windows

Accordingly to Apple support page:

If you have one of these Intel-based Mac models using OS X El Capitan or later, you don't need a USB flash drive to install Windows:

  • MacBook introduced in 2015 or later
  • MacBook Air introduced in 2017 or later3
  • MacBook Pro introduced in 2015 or later3
  • iMac introduced in 2015 or later
  • iMac Pro (all models)
  • Mac Pro introduced in late 2013 or later

# Find Boot Camp Assistant

Look for the Boot Camp Assistant app in your machine. It should be under Applications/Utilities folder.

# Run Boot Camp Assistant

This will fully install Windows 10 in your machine.
